Crear Entorno:
python -m venv yoloEnv

Activar entorno:
powershell: .\yoloEnv\Scripts\Activate.ps1
linux:      source yoloEnv/bin/activate


Instalar ultralytics en el entorno (icluye soporte para gpu):
pip install ultralytics[all]

Instalar torch con soporte de GPU:
pip uninstall torch torchvision torchaudio -y
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

Entrenar modelo:
yolo task=detect mode=train model=yolo11n.pt data="dataset/Evo detection.v3i.yolov11/data.yaml" epochs=1000 imgsz=640 device=0

Probar modelo:
yolo task=detect mode=val model=runs/detect/train/weights/best.pt data="yolo/dataset/Evo detection.v2i.yolov11/data.yaml" device=0
